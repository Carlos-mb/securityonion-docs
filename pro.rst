Security Onion Pro
==================

In 2022, we announced that we would be releasing enterprise features that would only be available to paid users of the platform. You can read the announcement at https://blog.securityonion.net/2022/08/security-onion-enterprise-features-and.html.

Starting in Security Onion 2.4.70, licensed users of Security Onion Pro can activate the following features.

.. note::

    Contact Security Onion Solutions, LLC via our website at https://securityonionsolutions.com for more information about purchasing a Security Onion Pro license to enable these features.

- :ref:`oidc` Support
  
  This feature allows you to configure 3rd-party authentication for :ref:`soc` using authentication providers such as Active Directory, Github, and Google. For more information, please see the :ref:`oidc` section.

- :ref:`luks` Disk Encryption

  Security Onion now supports several modes of :ref:`luks` disk encryption. For more information, please see the :ref:`luks` section. 

- :ref:`fips` OS Compliance
  
  FIPS is a requirement for many government customers. For more information, please see the :ref:`fips` section.

- :ref:`stig` OS Compliance

  STIG is a requirement for many government customers. For more information, please see the :ref:`stig` section.
  
- Time Tracking for :ref:`cases`

  This feature allows you to specify how much time you've spent working on a particular case and it will then show the total time spent by all analysts working on a particular case. For more information, please see the :ref:`cases` section.
